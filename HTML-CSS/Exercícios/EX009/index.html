Links para download

para adicionar um arquivo que você deseja que seja baixado no seu site, apenas adicione uma pasta e o arquivo na pasta do seu site. para que ele seja baixado ao clicar use o parâmetro "download=seu arquivo" dentro da tag a, e depois use o parâmetro "type=" e depois do sinal de igual você adiciona o tipo de arquivo. Cada arquivo tem uma forma diferente dentro desse parâmetro. As diferentes formas você pode encontrar no site iana.org
